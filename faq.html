Non. Chaque logiciel de la suite PLYIA est vendu avec une licence à vie. Vous payez une seule fois et le logiciel vous appartient pour toujours, mises à jour mineures incluses.

Les mises à jour de correction de bugs et d'amélioration mineures de l'interface sont gratuites. Des mises à jour majeures avec de nouvelles fonctionnalités significatives pourront être proposées comme des achats optionnels à un tarif préférentiel pour les détenteurs d'une licence.

Votre clé de licence personnelle vous permet d'activer le logiciel sur trois (3) de vos ordinateurs personnels. Par exemple, votre PC de bureau, votre ordinateur portable et un appareil sous un autre système d'exploitation.

Si vous achetez une version Windows et que vous possédez encore une activation de clé valide (sur les 3 disponibles), vous pouvez nous faire la demande pour recevoir également la version Linux sans frais supplémentaires.

Une connexion internet est requise uniquement pour trois actions initiales :

  1. La toute première activation de votre clé de licence.
  2. Le téléchargement des modèles d'IA recommandés via notre installateur.
  3. Le premier lancement du logiciel pour finaliser la configuration.

Une fois ces trois étapes passées, le logiciel est 100% fonctionnel hors ligne. Aucune donnée personnelle, d'utilisation ou de création n'est jamais envoyée vers nos serveurs. L'application ne recherche pas non plus les mises à jour automatiquement, garantissant qu'aucune connexion n'est établie sans votre accord.

Oui ! Nous ne vendons pas les modèles d'IA, qui sont des créations open source. Notre installateur télécharge et configure automatiquement le modèle open source que nous recommandons pour chaque logiciel car il est optimisé pour notre interface. Cependant, vous êtes absolument libre d'importer vos propres modèles compatibles (au format GGUF pour les LLMs, par exemple) pour une flexibilité maximale.

Chaque logiciel a ses propres prérequis. Nous avons mis en place un test de compatibilité interactif directement sur chaque page produit pour vous aider à vérifier si votre machine est adaptée. Nous vous encourageons vivement à l'utiliser avant tout achat.

Pour toute autre question, n'hésitez pas à nous écrire via notre page de contact. L'équipe PLYIA vous répondra dans les plus brefs délais.
