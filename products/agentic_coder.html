Agentic Coder

L'IDE intelligent 100% local, privé et open source. La puissance de l'IA agentique au service de votre code Python et HTML.

GRATUIT — Open Source Voir sur GitHub

Le premier produit gratuit de PLYIA. Agentic Coder est un projet communautaire destiné à démocratiser le développement assisté par IA locale.

Support Langages : Python (complet), HTML/CSS (aperçu)

Moteur IA : Llama.cpp (Format GGUF)

Configuration : 16 Go+ RAM | GPU NVIDIA recommandé (8 Go VRAM+)

🚀 100% Gratuit, 100% Open Source

Chez PLYIA, nous croyons que les outils fondamentaux de création doivent appartenir à ceux qui les utilisent. Agentic Coder est notre contribution à la communauté. C'est un logiciel sans télémétrie, sans licence payante et sans limites, conçu pour fonctionner là où vous êtes, même sans internet.

Prêt à coder en 3 étapes

1. Clonez le Projet

Récupérez le code source sur GitHub. L'installation est simple et ne nécessite qu'un environnement Python standard.

2. Importez vos GGUF

Glissez vos modèles favoris (Qwen2.5-Coder, Llama-3.2, etc.) au format GGUF dans le dossier models. Ils seront détectés automatiquement.

3. Codez avec l'Agent

Posez vos questions ou demandez des modifications complexes. L'agent analyse votre fichier et propose des Diffs intelligents.

L'IDE Local de Nouvelle Génération

Agentic Coder n'est pas qu'un simple éditeur de texte. C'est un environnement de développement qui intègre une intelligence agentique capable d'agir directement sur votre système de fichiers.

Pourquoi Agentic Coder ?

  • ✓ Zéro Abonnement : Pas de frais mensuels. Votre matériel est votre seul moteur.
  • ✓ Confidentialité Totale : Votre code ne quitte jamais votre machine. Idéal pour les projets sensibles.
  • ✓ Mode Diff Intelligent : L'IA remplace uniquement les lignes nécessaires, économisant ainsi du temps et des ressources GPU.
  • ✓ Terminal Intégré : Exécutez vos scripts Python et installez vos dépendances sans quitter l'interface.

💡 Fonctionnalités Avancées

Agentic Coder intègre des outils pensés pour les développeurs exigeants :

  • Indexation de Projet : L'IA a connaissance de l'arborescence complète pour mieux comprendre le contexte global.
  • Multi-Onglets : Travaillez sur plusieurs fichiers simultanément avec une interface fluide et moderne.
  • Auto-Correction : En cas d'erreur dans le terminal, un bouton vous permet de demander à l'IA d'analyser et de corriger le bug.

⚠️ Hardware et Inférence

Puissance Locale : Puisque Agentic Coder utilise llama.cpp pour l'inférence, la rapidité de génération dépend directement de votre processeur (CPU) et de votre carte graphique (GPU). Pour une expérience "agentique" fluide, l'utilisation d'un GPU avec au moins 8 Go de VRAM est fortement recommandée.

Modèles recommandés : Nous conseillons l'utilisation de modèles optimisés pour le code, comme Qwen2.5-Coder ou DeepSeek-Coder au format GGUF (quantification Q4_K_M ou supérieure).
